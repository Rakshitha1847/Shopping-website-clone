# Shopping-website-clone
A fully responsive Meesho clone built using only HTML and CSS with multiple interactive pages.

1. Design Features
   
Fully Responsive - Works on mobile, tablet, and desktop.

Meesho Color Scheme - Uses Meesho's signature pink (#ff3269) and clean white design.

Modern UI/UX - Clean layout with intuitive navigation.

Interactive Elements - Hover effects, transitions, and visual feedback.

2. E-commerce Features
 
Multi-page Navigation - 3 different pages with seamless transitions.

Product Catalog - Grid layout with product cards.

Category Browsing - Organized product categories.

Shopping Cart - Cart management with quantity controls.



3. Pages Included
 
Home Page - Landing page with hero section, categories, and featured products.

Products Page - Complete product listing with grid view.

Categories Page - Category-specific product browsing.

Cart Page - Shopping cart with item management.

4. Technologies Used

HTML5 - Semantic markup and structure.

CSS3 - Styling, animations, and responsive design.

Font Awesome - Icons for enhanced UI.

Google Fonts - Typography (Segoe UI).

CSS Grid & Flexbox - Layout and alignment.

CSS Variables - Consistent theming.
